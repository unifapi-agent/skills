# Influencer & creator discovery with UnifAPI

Find, score, and shortlist creators across Twitter / X, YouTube, TikTok, and Instagram — through one unified API. Live: [unifapi.com/solutions/creator-discovery](https://unifapi.com/solutions/creator-discovery).

## The agent loop

1. **Search** by niche, keyword, hashtag, or sound across platforms
2. **Profile** each creator — follower count, engagement rate, posting cadence
3. **Sample** their recent content for tone fit
4. **Score** with the LLM against the brand brief
5. **Hand off** the shortlist

## APIs used

- [Twitter / X API](https://unifapi.com/apis/twitter) — user lookup, recent tweets, follower stats
- [YouTube API](https://unifapi.com/apis/youtube) — channel info, recent uploads, view stats
- [TikTok API](https://unifapi.com/apis/tiktok) — user profile, video stats, hashtag search
- [Instagram API](https://unifapi.com/apis/instagram) — profile, posts, reels

## Why this works as a unified API

Per-platform integrations turn creator discovery into a backend project. UnifAPI lets a small team ship a creator-discovery agent in days: one bearer key, four endpoints, one normalized shape. Add an LLM scoring step and you have a product.
