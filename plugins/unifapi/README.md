Edit the root source files instead:

- `.codex-plugin/`
- `.mcp.json`
- `skills/`
- `docs/`

Then run:

```bash
node scripts/sync-distribution-bundles.mjs
```

Use `node scripts/sync-distribution-bundles.mjs --check` to verify that generated bundles have not drifted from the root source.
